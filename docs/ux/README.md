- Motion & animation standards

---

## Design Principles

### User-Centered
All design decisions prioritize student needs and learning goals.

### Accessible
WCAG 2.1 AA compliance minimum for all interfaces.

### Consistent
Reusable patterns and components ensure coherent experience.

### Scalable
Modular architecture supports future growth and feature additions.
