## Quick Checklist for Each Event

When adding an event to Sanity:

1. ‚úÖ Fill in all required fields (marked with \* in Sanity)
2. ‚úÖ Generate slug from title
3. ‚úÖ Set appropriate dates (future dates for website to display them)
4. ‚úÖ Choose correct event type
5. ‚úÖ Add tags (helps with filtering)
6. ‚úÖ Set target audience
7. ‚úÖ **Change Event Status to "Published"** (crucial!)
8. ‚úÖ Click the blue "Publish" button
9. ‚úÖ Wait for confirmation that it saved
10. ‚úÖ Refresh your website to see it appear

---

## Tips for Success

- **Dates:** Use future dates (January-February 2026) so events appear on the website
- **Descriptions:** Be specific and helpful - think about what students want to know
- **Capacity:** Office Hours and Hack Nights can be unlimited (leave blank)
- **Registration:** Virtual events often don't need registration; in-person events with food do
- **Status:** MUST be "Published" for events to show on website
- **Featured:** Mark 1-2 most important events as featured
